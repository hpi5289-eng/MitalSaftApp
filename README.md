# MITRAL INFOSYS - SAFT Android App

Staff Management App for MITRAL INFOSYS, Surat.

## Features
- Staff Login with Mobile/Password
- View Pending Works
- Mark Work as Complete

## API Endpoints
1. `android_login.php` - Staff Login
2. `get_pending_works.php` - Get Pending Works  
3. `mark_done.php` - Mark Work Done

## Setup
1. Upload `api/` files to your server
2. Update API URL in `LoginActivity.kt`
3. Build APK using GitHub Actions
